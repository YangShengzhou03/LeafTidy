## 为什么选择 LeafTidy？

### 完全离线，隐私安全

所有操作都在你的本地设备上完成，你的文件永远不会离开你的电脑。这意味着你不用担心隐私泄露，也不需要担心网络问题影响使用。

### 简单易用，上手即会

打开软件就能用，没有复杂的设置，没有学习成本。就像你的私人文件管家，安静、高效、安全地帮你打理好每一个文件。

### 操作可撤销，安心整理

自动记录每一次操作的详细日志，如果操作失误，随时可以找到并撤销，让文件恢复到原来的状态。

## 核心功能

### 文件整理

把散落在硬盘各个角落的照片、文档、视频自动整理到统一目录。支持按文件类型、日期、大小等多种规则整理，让每一个文件都能找到属于它的位置。

### 批量重命名

那些"IMG_20230520_143021.jpg"、"照片1.jpg"之类的名字让你根本不知道文件内容是什么？支持灵活的重命名模板，比如 `{日期}_{原名}`、`{序号}_{类别}`，一次操作就能给成百上千个文件起上清晰统一的名字。

### 重复文件清理

不同地方保存了相同的文件，占用了大量存储空间？使用 MD5 校验技术，基于文件内容进行精确的重复检测，让你安心清理，释放硬盘空间。

### 照片处理工具

照片是你最珍贵的数字资产，但它们常常存在各种问题：

- **照片脱敏**：移除 GPS 位置、相机型号等隐私信息，保护你的个人数据安全
- **地理位置分类**：基于离线地名库，按省/市/区县自动分类照片
- **AI 智能分类**：自动识别照片内容并分类（开发中）
- **拍摄时间修复**：修正照片错误的拍摄日期（开发中）
- **GPS 写入**：为照片添加地理位置信息（开发中）

### 系统清理

清理 Thumbs.db、.DS_Store、desktop.ini 等系统垃圾文件，让你的文件夹更干净。

## 如何使用？

### 第一步：下载安装

访问 [Releases](https://github.com/YangShengzhou03/LeafTidy/releases) 页面下载 Windows 安装包，双击安装即可。

### 第二步：选择功能

打开软件后，从左侧导航栏选择你需要的功能，比如文件整理、批量重命名、重复清理等。

### 第三步：添加文件

点击添加按钮或直接拖拽文件到应用中，设置整理规则、重命名模板等参数。

### 第四步：预览确认

查看操作预览列表，确认无误后再执行。如果不确定，可以先小范围测试。

### 第五步：执行操作

点击执行按钮，LeafTidy 会自动完成整理工作。

### 第六步：查看日志

如需回滚，从日志页面找到对应操作即可撤销。

## 常见问题

### 是否支持其他操作系统？

目前仅支持 Windows。macOS 和 Linux 版本正在规划中。

### 是否需要联网？

不需要！所有操作都在本地完成，完全离线运行。

### 数据是否安全？

非常安全！你的文件永远不会上传到任何服务器，所有处理都在你的电脑上完成。

### 操作失误怎么办？

所有操作都有详细日志，可以随时查看和撤销，让你的文件整理过程更加安心。
